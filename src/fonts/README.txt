Seluruh web WAJIB menggunakan font Google VT323. Semua komponen, halaman, dan elemen harus mewarisi font ini.

Implementasi:
- Import VT323 dari Google Fonts di layout.tsx
- Terapkan font-family ke body/html/global Tailwind
- Semua komponen harus menggunakan font ini.

Catatan: Jika menambah komponen baru, pastikan tetap mengikuti aturan ini.
